$playlist

All playlist commands.

add

Add a song(s) to a playlist.

$playlist add -p [playlist_id] -s [song_id] -s [song_id]

Examples:

Add one song to a playlist. $playlist add -p 2 -s 74
Add more than one song to a playlist. $playlist add -p 2 -s 74 -s 66 -s 12
